Projects

TrueStock
TrueStock is an inventory management system developed using Java, designed to help small to medium-sized businesses efficiently track and manage their inventory.
IruBasha
IruBasha is a kids' learning app primarily designed to support Sinhala and Tamil translation and vocabulary building. The app features a simple, interactive interface with:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Prolog
- Database: SQL integration
- AI integration for improved language learning support

It aims to make bilingual learning fun and accessible for young children.
MedGuide
MedGuide is an assistance website designed to help users extract key information from medical scans, tests, and reports. It also provides conversational support to help users understand the seriousness of their condition in simple terms.
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
MedGuide aims to make medical data easier to understand for everyone.
